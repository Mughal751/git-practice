Flexbox — about layout

Hey! I'm a developer in the Dev Weekend Summer Fellowship. This page is my Week 2 task deliverable — a static HTML/CSS layout that demonstrates the four core CSS concepts: flexbox, positioning, the box model, and CSS grid.

Every section of this page was written by hand to practice these concepts. The navbar uses flexbox. The project cards use CSS Grid. The badges are absolutely positioned. The spacing throughout uses margin and padding from the box model.

The goal is to understand why each property works, not just what it does — so every concept has a comment in the source code explaining the rule behind it.
